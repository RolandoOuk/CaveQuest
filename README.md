# CaveQuest - Beta Version
A 2D platformer developed in Godot 4.5.1. This beta version features a playable level with physics-based movement, hazards, and a win condition.
## Features
* **Character Controller:** Smooth movement and jumping physics.
* **Hazards:** Spikes and water that trigger a level restart.
* **Goal System:** A functional finish line that ends the game.
